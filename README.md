# Paperless-Conference-System

## backend set up

- cd backend
- npm install
- npm install mysql2
- npm install cors
- npm install dotenv

- create .env file, isi configurasi database

### running backend

- create table : npm run migrate
- run be : npm start

## frontend set up

- cd ../frontend
- npm install
- npm install react-router-dom

### running frontend

- npm run dev
